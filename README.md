# SAÉ S2.04 (Data Visualization)

The file `data.csv` located at the root contains all the data used for the 7 exercises.

For each student, the file contains the following attributes:

* **Student_ID**
* **Age**
* **Gender**
* **Academic_Level**
* **Country**
* **Avg_Daily_Usage_Hours**
* **Most_Used_Platform**
* **Affects_Academic_Performance**
* **Sleep_Hours_Per_Night**
* **Mental_Health_Score**
* **Relationship_Status**
* **Conflicts_Over_Social_Media**
* **Addicted_Score**
